# RSS Feeds

## Releases RSS Feed

The `RSSVersionFeed` class controls the RSS feed for new Boost releases.

**URLS**:

- `/feed/downloads.rss` to replicate the original URL
- `/feed/downloads.atom` to publish an Atom version
